# 🛒 E-Commerce Sales Dashboard

## Project Overview
End-to-end data analysis project using a messy e-commerce dataset.
Covers data cleaning, exploratory analysis, and interactive dashboard building.

## Tools Used
- Microsoft Excel — data cleaning
- Power BI Desktop — dashboard and visualization

## Dataset
- 103 orders with intentional data quality issues
- Columns: Customer, Product, Category, Price, Quantity, Total, Date, Payment Method, Status

## Data Issues Fixed
- Missing values in Price, Quantity and Total columns
- Inconsistent category naming (sports, ELECTRONICS, electronic)
- Invalid price values (abd, four hundred, 300$)
- Negative quantities and totals (corrupted rows)
- Inconsistent date formats

## Dashboard Features
- 3 interactive views using bookmarks and buttons
- Sales Overview — KPI cards, revenue trend, category breakdown
- Products & Orders — top products, payment method analysis
- Details — filterable full orders table with slicers
